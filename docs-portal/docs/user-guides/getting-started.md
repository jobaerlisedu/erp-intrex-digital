# Getting Started Guide

Welcome to the Intrex ERP & CRM platform. This guide helps new operators onboard, navigate the interface, and understand basic operational workflows.

---

## 1. Accessing the Platform

To access the ERP:
1. Open your web browser and navigate to the application address:
   *   **Local URL**: `http://localhost:8000/` (for local sandbox environments).
   *   **Production URL**: `https://erp-intrex-digital.onrender.com` or your company's custom subdomain.
2. Click the **Login** link in the navigation menu.
3. Input your username and password, then click **Sign In**.

> [!NOTE]
> If you do not have an account, contact your system administrator. Your administrator will create your profile and assign you to the correct module groups.

---

## 2. Dashboard Layout

Once logged in, you will be redirected to the main **ERP Dashboard**. The interface is structured as follows:

```
┌────────────────────────────────────────────────────────┐
│  Intrex Digital ERP                 [User Profile] [🔒] │
├───────────────┬────────────────────────────────────────┤
│  Sidebar      │  Main Content Area                     │
│               │                                        │
│  * HRM        │  Module Summary Widgets                │
│  * Billing    │  * Employee Count: 48                  │
│  * Inventory  │  * Active Projects: 12                 │
│  * EdTech     │  * Invoices Outstanding: $8,400        │
│  * Projects   │                                        │
└───────────────┴────────────────────────────────────────┘
```

*   **Top Header**: Contains the platform branding, quick profile links, and the logout icon.
*   **Navigation Sidebar**: Displays the list of modules you are authorized to access. If a module is missing, it means your account lacks the necessary Group permission.
*   **Main Content Panel**: Houses the widgets, tables, and workflows associated with the active module.

---

## 3. Basic Workflows

### A. Navigation
Clicking any module name in the sidebar loads the default dashboard view for that section. Use the submenu tabs or primary buttons on each page to trigger actions (e.g. creating requisitions, printing pay slips).

### B. Logging Out
For security compliance:
1. Click the **Logout** button (door icon) in the header.
2. The system terminates your session immediately, logging you out across all active tabs.

---

## 4. Password Security

It is recommended to update your temporary password after logging in for the first time:
1. Navigate to **User Profile** (or ask an administrator).
2. Input your new password in the profile settings form.
3. Click **Update Profile** to synchronize the credentials.
